ITSM : Action utilisateur requise
Vous venez de recevoir une notification par mail : Votre action est ncessaire
1. Accdez sur la page d'accueil d'ITSM via le lien se trouvant sur Easy Internet :
2. Cliquez sur le lien " Voir mes incidents en cours " :
3. La liste de vos incidents en cours est affiche :
1 Numro identifiant votre ticket
2 Statut actuel de votre ticket
Dans ce cas : Action utilisateur
4. Le technicien vous demande des renseignements complmentaires ou d'effectuer une action.
5. L'action a rsolu le problme et vous pouvez travailler nouveau normalement
Cliquez sur le bouton : Rsolution
Cette action clture l'incident "sans suite"
6. Le problme n'est pas rsolu et l'incident persiste :
Indiquez ce qui ne va pas dans le champs : Commentaires additionnels
Cliquez sur le bouton : Mettre jour
