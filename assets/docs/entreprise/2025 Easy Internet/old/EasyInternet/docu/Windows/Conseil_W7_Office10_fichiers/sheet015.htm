Ajout d'une connexion Pulse Secure
Ce logiciel remplace le pack Business Everywhere d'Orange et Junos Pulse
Il permet donc de se connecter au rseau Wienerberger.
1. Cliquez sur le bouton "Dmarrer" puis "Tous les programmes"
Ouvrez le rpertoire Pulse Secure \ Pulse Secure
Cliquez sur le programme Junos Pulse
2. Cliquez sur la crois d'ajout de connexion
3. Saisir dans le champ "Nom :"
wbuser_XXX
remplacer Wbuser_XXX par votre identifiant
4. Saisir dans le champ "Adresse URL du serveur :"
https://wienerberger.bvpneu.com/core
5. Cliquez sur "Connexion"
6. Saisir dans le champ "Nom d'utilisateur :"
wbuser_XXX@wb-fr-beu.fr.op
remplacer Wbuser_XXX par votre identifiant
7. Saisir votre mot de passe dans "Mot de passe : "
8. Cochez la case "Enregistrer les paramtres"
9. Cliquez sur "Connexion"
10. Une fois la connexion tablie, Junos Pulse vous affiche ce message :
11. Si ce message apparat :
Ressaisissez vos identifiants
Vrifiez que votre partage de connexion Iphone fonctionne
