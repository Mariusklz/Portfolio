Gérer le Volet de Navigation
Le volet de navigation fait partie intégrante de l'exploreur Windows. 
Vous pouvez ouvrir ou fermer comme bon vous semble les volets avec les flèches à gauche des icônes.
Le double clique sur les volets fonctionne également. 
Sous les "Favoris" du volet de navigation, vous retrouvez : 
Bureau : Tous les fichiers/répertoires se trouvant sur votre Bureau Windows
Emplacements récents : Les répertoires que vous avez récements ouverts
Téléchargements : Lieux où se déverse les téléchargements par Internet Explorer 
Accéder à tous vos documents en ouvrant le volet "Bibliothèques" 
Accéder aux Lecteurs Réseaux en ouvrant le volet Ordinateur
