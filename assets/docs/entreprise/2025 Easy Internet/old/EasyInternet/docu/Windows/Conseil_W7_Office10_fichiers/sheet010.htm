Indicateur d'absences Outlook
Pour modifier votre indicateur d'absence, allez dans le menu "Fichier" en haut gauche.
Cliquez sur le bouton "Rponses automatiques"
Cochez la case "Envoyer des rponses automatiques"
Veillez bien indiquer le message dans les deux options :
- "Au sein de mon organisation" ==> Membre de la socit Wienerberger (EINSTEIN)
- "En dehors de mon organisation" ==> Toute personne trangre la socit Wienerberger (non-EINSTEIN)
