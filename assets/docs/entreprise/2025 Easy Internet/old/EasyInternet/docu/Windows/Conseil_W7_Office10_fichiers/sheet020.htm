ITSM : Solution proposée
Vous venez de recevoir une notification par mail : Proposition dune solution 
1. Accédez sur la page d'accueil d'ITSM via le lien se trouvant sur Easy Internet :
2. Cliquez sur le lien " Voir mes incidents en cours " : 
3. La liste de vos incidents en cours est affichée : 
Dans ce cas :  Solution proposée
4. Le technicien vous a proposé une solution à votre problème.
5. Le problème est bien résolu et vous pouvez travailler à nouveau normalement
Cliquez sur le bouton : Fermer l'incident 
6. Le problème n'est pas résolu et l'incident persiste  : 
Indiquez ce qui ne va pas dans le champs : Commentaires additionnels
Cliquez sur le bouton : Rouvrir l'incident
