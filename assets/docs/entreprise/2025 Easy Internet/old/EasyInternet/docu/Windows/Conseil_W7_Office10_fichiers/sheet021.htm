ITSM : Attacher une pièce jointe ou une image
Vous voulez joindre une copie d'écran ou un fichier à un INCIDENT
Saisissez d'abord le texte de votre ticket.
1. Pour attacher des pièces jointes (en mode création de ticket)
Cliquez sur le trombonne
Cliquez sur "Parcourir" pour sélectionner le fichier à joindre puis sur "joindre" pour attacher la pièce sélectionnée.
Vous pouvez ajouter une autre pièce en cliquant sur "Ajouter une autre pièce jointe"
Les pièces jointes apparaissent dans le message : 
2. Pour insérer une image du presse papier, cliquez sur "Paste Image" (en mode création de ticket)
cliquez sur "Paste Image"
Collez l'image du presse-papier par Ctrl-V et cliquez sur OK ;
Il se peut que l'image n'apparaisse pas immédiatement dans votre ticket,mais elle y est bien jointe.
3. Quand vous retournez dans un ticket, la procédure reste la même. 
L'icone "attachement" reste la identique, mais pour coller une image du presse papier, l'icone est une image avec une montagne.
