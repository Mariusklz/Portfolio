Connexion Pulse Secure
Ce logiciel remplace le pack Business Everywhere d'Orange et Junos Pulse
Il permet donc de se connecter au rseau Wienerberger.
1. Cliquez sur le bouton "Dmarrer"
2. Lancez le logiciel Pulse Secure
3. Cliquez sur le bouton "Connexion" pour vous connecter au rseau Wienerberger.
Le statut passe "Connect" aprs quelques secondes
4. Vous pouvez fermer PULSE(bouton "Fermer"). La connexion reste tablie.
Vous pouvez dsormais accder aux diffrents lecteurs rseaux (FR_Mtiers, EasyInternet, CRM...)
5. Pour se dconnecter du rseau Wienerberger, relancez Pulse Secure
Cliquez sur le bouton Dconnexion. Le statut redevient  Dconnect .
