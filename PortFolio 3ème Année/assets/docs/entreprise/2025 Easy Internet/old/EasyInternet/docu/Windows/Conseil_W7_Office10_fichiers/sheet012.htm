Modifier une pièce jointe
Un fichier en pièce jointe ouvert dans un email réceptionné, est ouvert en "Lecture seule".
Attention, les modifications sont possibles, mais l'enregistrement ne se fera pas.
Il existe deux méthodes de modifications de pièces jointes, selon si vous souhaitez "Transférer" ou "Répondre" au mail : 
Lorsque vous souhaitez modifier un fichier en pièce jointe, pour le transférer, suivez ces étapes : 
Ouvrez l'email en lecture, puis cliquez sur "Transférer".
Le fichier est alors ouvert en "Mode protégé", cliquez alors sur "Activer la modification".
Une fois vos modification effectuées, "enregistrez" le fichier avec la disquette et fermez le fichier.
La pièce jointe modifiée sera alors transmise lorsque vous cliquerez sur "Envoyer"
Lorsque vous souhaitez modifier une pièce jointe pour répondre au mail, suivez ces étapes : 
Ouvrez l'email en lecture, faites un clique droit sur la pièce jointe, puis cliquez sur "Enregistrer sous".
La fenêtre d'enregistrement Outlook s'ouvre : 
1) Choisissez où vous allez enregistrer votre pièce jointe pour l'ouvrir par la suite.
Par défaut, Outlook enregistrera dans vos "Documents".
2) Modifiez le nom de votre pièce jointe si vous le souhaitez.
3) Enregistrez votre pièce jointe.
Ouvrez la pièce jointe que vous venez d'enregistrer, à l'endroit où vous avez choisi.
Effectuez les modifications et enregistrez votre document.
Dans le mail de départ, avec lequel vous avez enregistré la pièce jointe, cliquez sur "Répondre".
Dans la réponse au mail, cliquez sur "Insertion" puis sur "Joindre un fichier".
La fenêtre d'insertion de fichier s'ouvre : 
1) Rechercher la pièce jointe que vous venez de modifier. 
2) Cliquez sur le fichier à insérer.
3) Cliquez sur "Insérer".
La pièce jointe est alors attaché à votre réponse au mail. 
Il ne vous reste plus qu'a taper votre texte dans le mail puis cliquer sur Envoyer.
