Modifier ma signature Outlook
Allez dans le menu "Fichier" en haut gauche. Puis cliquez sur "Options".
Dans la fentre qui s'ouvre, cliquez sur le menu "Courrier", puis sur le bouton "Signatures"
Une autre fentre s'ouvre dans laquelle vous pouvez modifier votre signature.
Veillez ce que le titre de votre signature appart bien dans "Nouveaux messages" et "Rponses/transferts" comme ci-dessus.
Veillez galement respecter la charte graphique Wienerberger. Une note est disponible au service marketing.
